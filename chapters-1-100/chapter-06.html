6.1 Introduction

The 30+ system is a classic three‚Äëcushion system mainly used in positions where the cue ball departs via a long rail and must reach the third rail on or around the long rail. The system is particularly suitable for shots where the second ball lies relatively free and the trajectory can be well controlled.

The name ‚Äú30+‚Äù refers to the base value in the calculations. This chapter covers the basic formula, the reference points, and typical patterns in which this system works very reliably.

6.2 Basic Formula of the 30+ System

The core of the 30+ system is a simple calculation relationship between departure point, arrival point, and aiming point. A commonly used basic form is:

‚û§ Aim point = 30 + (Departure point ‚Äì Arrival point)

This formula is a guideline and must be corrected depending on:

  • table speed (new or old cloth)
  • amount of spin
  • precision of the departure point
  • position of the second ball

The better the player knows the table, the more reliable this formula becomes in practice.

6.3 Reference Points and Calculations

The reference points for the 30+ system are based on fixed values on the long and short rails. The player first determines the departure point of the cue ball, then the desired arrival point on the third rail.

The values are usually structured as follows:

  • Departure point ‚Äî value along the long rail where the cue ball departs
  • Arrival point ‚Äî value of the target area on the third rail
  • Aim point ‚Äî value on the first rail to aim at

By placing these three values into the basic formula, the player gets an initial guideline for the correct line and can then adjust with spin and speed.
